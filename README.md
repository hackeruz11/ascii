## 📌 Qisqacha

**ASCII Media Studio** - bu brauzerda ishlaydigan kuchli vosita. Hech qanday server yoki internet kerak emas (YouTube dan tashqari). Siz:

- 📹 **Videolarni** jonli ASCII san'atiga aylantirasiz
- 🖼️ **Rasmlarni** rangli yoki oq-qora ASCII ga o'tkazasiz
- 🌍 **YouTube videolarini** thumbnail orqali ASCII qilasiz
- 📸 **Kameradan** suratga olib, darhol ASCII ko'rasiz
- 💾 **Barcha natijalarni** TXT fayl sifatida saqlaysiz

## 🚀 Qanday ishlatiladi

### 1. Videoni ASCII ga o'tkazish

```bash
1. "VIDEO" menyusiga kiring
2. Video faylni tanlang yoki URL kiriting
3. Kenglikni tanlang (200px eng aniq)
4. "ASCII ga o'tkazish" tugmasini bosing
5. Video harakatli ASCII sifatida ijro etiladi
```

2. Rasmdan ASCII yaratish

```bash
1. "RASM" menyusiga kiring
2. Rasm faylni tanlang
3. Usulni tanlang (Rangli / Oq-qora)
4. Kenglikni sozlang (200px eng aniq)
5. "ASCII ga o'tkazish" tugmasini bosing
6. "Saqlash" tugmasi bilan TXT fayl yuklab oling
```

3. YouTube dan ASCII yaratish

```bash
1. "YOUTUBE" menyusiga kiring
2. YouTube havolasini kiriting (youtu.be yoki youtube.com)
3. Kenglikni tanlang
4. "Yuklash" tugmasini bosing
5. Video thumbnail ASCII san'ati yaratiladi
```

4. Kamera bilan suratga olish

```bash
1. "KAMERA" menyusiga kiring
2. "Suratga ol" tugmasini bosing
3. Ruxsat bering va suratga oling
4. ASCII san'at avtomatik yaratiladi
5. "Saqlash" tugmasi bilan faylni saqlang
```
